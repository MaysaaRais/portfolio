# Portfolio Look-Through & Concentration Risk

A risk management framework for Fund-of-Funds and multi-asset mandates. This project mimics institutional "look-through" logic to identify hidden exposures.

### Implementation Focus
- **Concentration Tracking:** Aggregating underlying GP data to monitor exposure across Sector, Geography, and Vintage Year.
- **Mandate Compliance:** Flags for when a portfolio drifts from its Strategic Asset Allocation (SAA).
- **Visualization:** Heatmaps and breakdown charts for institutional risk reporting.
