# Volatility Forecasting: GARCH vs RNN

Comparison of econometric and deep learning methods for predicting market volatility regimes.

### Implementation Focus
- **GARCH(1,1):** Modeling volatility clustering using Maximum Likelihood Estimation (MLE).
- **Recurrent Neural Networks:** Implementing a custom RNN to capture non-linear risk patterns in daily returns.
- **Backtesting:** Evaluating forecasting accuracy (MSE/MAE) across different market cycles.
