# LBO Scenario Simulator (Monte Carlo)

Stochastic approach to LBO modeling. Instead of static Excel cases, this simulates 1,000+ scenarios to find the probability of reaching target returns.

### Implementation Focus
- **Core LBO Logic:** Modeling debt paydown, interest expense, and the equity waterfall.
- **Stochastic Variables:** Monte Carlo simulation on exit multiples, EBITDA growth, and margin improvements.
- **Outcome Analysis:** Probability density plots for IRR and MOIC to identify downside break-even points.
